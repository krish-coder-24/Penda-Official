Installation 
------------------

Kali Linux & Termux & Debian
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

| 1) **Step 1**
.. code-block:: sh

   git clone https://github.com/krish-coder-24/Penda-Official

| 2) **Step 2**

.. code-block:: sh

  cd Penda-Official

| 3) **Step 3**

.. code-block:: sh

   bash penda-install

| One - Line Installation 
+++++++++++++++++++++++++

.. code-block:: sh

   git clone https://github.com/krish-coder-24/Penda-Official && cd Penda-Official && bash penda-install 


Windows
^^^^^^^
